## كيفية البدء

1. **الخطوة 1**: قم بتثبيت Node.js و npm.
2. **الخطوة 2**: قم بتثبيت الحزم اللازمة عبر npm باستخدام الأمر:
   ```bash
   npm install
   ```
3. **الخطوة 3**: قم بتشغيل الخادم عبر:
   ```bash
   npm start
   ```
